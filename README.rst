What is Aho?

`Aho` is the Application Calling Script for the Mac OSX users.

the following conditions required:

1) You have the following path on your MacOSX:

      /Applications/

2) Ruby installed


Usage
-------------------------------

1) First Install the program.

     $ ./setup

2) Secondly, type as follows:

     aho help
